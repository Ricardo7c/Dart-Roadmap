# Sintaxe Basica

### 01 - Olá Mundo

**Enunciado:**
Crie um programa que exiba “Olá, Mundo!” no terminal.

**Objetivo:**
Familiarizar o aluno com a estrutura básica de um programa Dart e o comando `print`.

**Requisitos de Conhecimento:**
`main()`, `print()`

**Exemplo de Saída no Terminal:**

```
Olá, Mundo!
```
